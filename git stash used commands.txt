git stash
git stash save 1 (добавил(сохранил) коммит в индексе 0)
git stash list (проверил список всех коммитов скрытых)
git stash apply 0 (внёс измененния из коммита и вернул их назад в рабочий режим из сокрытия)
git stash drop 0 (задискардил все измененния из стэша)
git stash pop 0 (задискардил все изменнения при этом заээплая их)
git stash --index
